# Character asset sources

- `rpg/`: Quaternius, [RPG Character Pack](https://quaternius.com/packs/rpgcharacters.html), November 2020. Six rigged, animated fantasy character archetypes distributed as self-contained glTF files.
- License: [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/). The models may be used, modified, and redistributed in personal and commercial projects.

The game preserves its deterministic race, identity, profession, palette, equipment, and multiplayer appearance layers on top of these archetypes. Only the runtime files used by the six player classes are included.
