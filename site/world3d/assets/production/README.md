# Sanctum & Shadow production visuals

These GLB files are generated by `tools/blender/build_visual_library.py` and are runtime assets for the game.

- Buildings, interiors, vegetation, props, weapons, materials, and animation clips are original project-generated work built from Blender primitives.
- Character topology and the base armature derive from the CC0 Quaternius animated character documented in `site/prototype/ASSET-LICENSE.md`.
- The editable source library is `tools/blender/sanctum-visual-library.blend` and is kept outside this browser-served directory.

Regenerate the complete package with Blender 5.2 or later using the command in `tools/blender/README.md`.
